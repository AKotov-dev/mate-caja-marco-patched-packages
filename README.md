# mate-caja-marco-patched-packages
## What is this

This repository contains patched packages for caja and marco (MATE Desktop Environment).

The modifications are minimal and applied at build time via spec files
(e.g. small sed-based changes to upstream sources).

## Changes
- caja: disabled trust/security warning dialogs for desktop launchers
- marco: reduced window and menu shadow opacity
